What's been done:

Description of Application: 
- Captured real-time audio segmented into 10-second intervals using iOS AVFoundation kit and connected it via Storyboard
- Utilized multithreading to send audio snippets to a specialized LLM for insight extraction without interrupting live audio capture
- Adopted Deepgram for speech-to-text conversion and Core Data for storing session data


Features implemented:
	Conversation History with search.
	Windows for Trnascript, Q&A + Model Thoughts.
	Windows are Integrated temporarily with OpenAi.
	Chats names are editable, storage via core data.
	Microphone for Live Transcript
	Integration with Backend.
	UI Overhaul
